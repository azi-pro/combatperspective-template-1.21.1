本目录包含从 [Minecraft Wiki](https://minecraft.wiki) 下载的官方文档。

## 内容结构

```
docs/minecraft-wiki/
├── README.md              # 本文档
├── Java_Edition/         # Java Edition 核心文档 (32 页)
│   ├── Data_pack.html     # 数据包格式
│   ├── Advancement.html   # 进度系统
│   ├── Recipe.html       # 配方系统
│   ├── Loot_table.html   # 战利品表
│   ├── Tag.html          # 标签系统
│   ├── Resource_pack.html # 资源包
│   ├── Commands.html     # 命令参考
│   ├── Function.html     # 函数
│   └── ...
└── Related/             # 相关页面
    ├── Java_Edition.html
    ├── Java_Edition_development.html
    └── ...
```
